# RTGD-Engine
Just a simple engine. Engine uses thirdparty solutions: 
- [flecs](https://github.com/sandermertens/flecs) framework for ECS
- [enkiTS](https://github.com/dougbinks/enkits) as a task scheduler
- [assimp](https://github.com/assimp/assimp) for mesh loading
- [gainput](https://github.com/jkuhlmann/gainput) for handling input
- [spdlog](https://github.com/gabime/spdlog) for logging
- [stb image](https://github.com/nothings/stb) for image manipulations
- [DiligentEngine](https://github.com/DiligentGraphics/DiligentEngine) as rendering abstraction layer 

RTGDEngine supports both linux and windows.
